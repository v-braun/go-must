## Installation
```sh
go get github.com/v-braun/go-must
```



## Usage

``` go


must.ArgNotNil(val, "val") // if val == nil will panic with "argument val is nil"

must.ArgBeValid(val > 1, "val invalid: (%d > 1) == false", val) // if val <= 1 will panic with val invalid: (0 > 1) == false

```
